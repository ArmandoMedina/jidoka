# Ruta sugerida

Para trabajo diario:

`00` → `01` → `10` → (`17` si no hay trabajo definido) → (`11` si el QUÉ está borroso) → `12` → `13`/`18` → `14` → `15` → `16`.

Para adopción o mantenimiento:

`00` → `02` → `30`, `40`, `41`, `42` o `44`.

Para auditoría y aprendizaje:

`00` → `03` → `70`, `71` o `72`.

Para publicar Jidoka:

`00` → `04` → `81` → `80`.
