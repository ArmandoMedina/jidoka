# Sprints — el índice-récord

> Una fila por sprint, con el **estado real** (no el deseado). La columna **`#` es el orden canónico (cronológico)** del linaje — el número del sprint; la etiqueta conserva su nombre histórico. El plan aprobado se archiva como `sprint-<slug>-plan.md`; el récord de cierre como `sprint-<slug>-entrega.md` (plantillas en [`kit/.jidoka/templates/`](../../kit/.jidoka/templates/)). Los `cierre-*` son cierres de sesión sin sprint formal (marcados con `·`).
